## Trader Behavior vs Market Sentiment Analysis

This project analyzes how trader behavior changes under different market sentiment conditions (Fear vs Greed) using historical trading data and Bitcoin market sentiment data.

### Objective
To understand how profitability, leverage, volume, and risk exposure align with overall market sentiment and to identify behavioral patterns that can support smarter trading strategies.

### Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

1. Open notebook_1.ipynb in Google Colab
2. Upload the provided CSV datasets
3. Run all cells sequentially
4. Generated CSV files and visual outputs will be stored in their respective folders
